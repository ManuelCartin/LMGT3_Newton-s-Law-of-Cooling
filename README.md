# LMGT3_Newton-s-Law-of-Cooling
# 🧊 Newton’s Law of Cooling Applied to LMGT3

## 📌 Project Description

This project aims to apply Newton’s Law of Cooling to a realistic motorsport context, specifically in the LMGT3 category, to analyze the thermal behavior of vehicle components (such as brakes and tires) during racing conditions.

It seeks to demonstrate how classical physics principles can be integrated with data analysis and simulation to generate relevant insights into automotive performance.

---

## 🎯 Objectives

* Apply Newton’s Law of Cooling to simulated or real data.

* Model the temperature variation in components of an LMGT3 car.

* Analyze how factors such as speed, environment, and time affect cooling.

* Integrate physics concepts with data science tools.

---

## 🧠 Theoretical Framework

Newton's Law of Cooling states that:

> The rate of change of an object's temperature is proportional to the difference between its temperature and the ambient temperature.

It is expressed as:

```
dT/dt = -k (T - T_ambient)
```

Where:

* `T` = object temperature
* `T_ambient` = ambient temperature
* `k` = cooling constant
* `t` = time

## 🏎️ LMGT3 Context

LMGT3 category cars, used in endurance races, present extreme conditions where thermal control is key:

* Tires depend on optimal temperature ranges

This project simulates or analyzes these behaviors using mathematical models.
